# 2021Networks

Code in support of numerical optimization supporting the paper

Martineau, S., Saffold, T., Chang, T. T., & Ronellenfitsch, H. (2022). Enhancing synchronization by optimal correlated noise. Physical Review Letters, 128(9). https://doi.org/10.1103/physrevlett.128.098301

and a project analyzing and simulating a grid frequency control system using ESOP grid frequency data.
